# unocss-preset-antd

(incomplete and untruthful) port of [ant-design](https://github.com/ant-design/ant-design) for [unocss](https://github.com/unocss/unocss)

```sh
# development
pnpm i
pnpm dev-setup
pnpm dev

# publish demo app
pnpm build
pnpm release-production

# publish unocss-preset-antd
pnpm build-lib
pnpm release-lib
```
